环境

  • MIUI 13.22.4.7 开发版
  • Magisk 25.2 (25200)
  • Fake Location 1.3.1.8 (1572)
  • ZDTY 2.9.3

这里我用了 Magisk 来获取 ROOT / 安装模块，因为我其实也只是个小白 /(ㄒoㄒ)/~~ 用 TWRP 等方案的大佬我就无能为力了，不过思路应该是一样的。

关于 ZDTY 的版本，这是截至目前（2022/11/30）的最新版，这几天测试下来非常稳定，后续会不会再更新更多反作弊检测手段未知。2.8 及更早的版本不会检测虚拟机环境，经同学测试可以直接用 vmos，但不知道什么时候就会被强制更新，未雨绸缪，故有此文。

准备

获取 ROOT 权限

  • 解锁 Boot Loader，各品牌方法不同，请自行搜索。再次强调，这会带来不可逆的后果，请自行考虑清楚！
  • 安装 Magisk。
  • 找到自己手机系统的刷机包，提取 boot.img，用 Magisk 修补后，再用 fastboot 刷入手机。

以上过程写得非常简略，因为我觉得没什么必要，也可以视为一个小小的门槛吧。留给读者作为习题

隐藏 ROOT 权限

由于 ZDTY（以及绝大部分支付类 APP）会检测 ROOT 权限，因此我们要隐藏它。Magisk 自版本 24 以后去除了原生隐藏 ROOT 支持，这里用 Shamiko 模块来实现。

  • 下载 Shamiko，这是一个 zip 文件。在 Magisk 的 模块 选项卡中选择 从本地安装，然后选择刚刚下载的 zip 文件。
  • 在 Magisk 右上角 设置 中打开 Zygisk，重启。
  • 在 Magisk 设置中找到 配置排除列表，选择 ZDTY，点击展开，将所有条目选中，然后勾选之。顺便可以将其他 APP 也勾选。注意不要打开 遵守排除列表。正确的操作后 Shamiko 应该提示😋
  • 打开 ZDTY，如果没有弹窗索取 ROOT 权限则已完成。

配置 Fake Location

  • 安装 Fake Location，注册账号，购买专业版授权。
  • 将 运行模式 改为 ROOT模式，初次设置会索取 ROOT 权限，请自行决定是否继续。
  • 切换到 路线模拟，点击右侧+号来新建一条路径，完成后打开 循环，将 速度 改为 6m/s，并打开 速度浮动 和 步频模拟，并将 步频 改为 2.5 步 /s。
  • 切换到 设置，并按下图设置。

隐藏 Fake Location

Fake Location 自带了隐藏 APP & 隐藏 ROOT，但并不能骗过 ZDTY。测试发现 ZDTY 检测 Fake Loation 的方式是调用系统 API 来查找其包名com.lerist.fakelocation，那就简单了，ban 掉就好。

  • 安装 LSPosed 框架，下载的时候记得选择 zygisk release，别搞错了。

  • 切换到 Magisk，用和 Shamiko 相同的方式从本地安装。

  • 回到 LSPosed，确保激活状态是 已激活。切换到最左侧 仓库 选项卡，搜索 隐藏应用列表，点开，切换到 版本，拉到最底下点击下载 附件，然后直接安装刚刚下载的 apk 文件。

  • 回到 LSPosed，切换到 模块 选项卡，选择 隐藏应用列表，点击右上角省略号，勾选推荐，然后 启用模块。

  • 打开 隐藏应用列表，确保已激活，然后点击 安装 Magisk 插件，选 zygisk，完成后重启。

  • 点击 模板管理，创建黑名单模板，在 隐藏应用 列表把拥有 ROOT 权限的 APP 全勾上，包括 Magisk、Fake Location 等等。然后在 maps 规则 列表中添加以下字段：

    magisk
    root
    lsposed
    org.lsposed.manage
    com.topjohnwumagisk
    magisk manager
    com.lerist.fakelocation
    fakelocation
    lerist

  • 完成后点击右上角 保存，回到插件主页，点击 选择生效应用，选中 ZDTY，启用隐藏，勾选 启用所有隐藏方式，并启用刚刚配置好的模板，点击右上角 保存。

至此，所有准备工作都已经完成。

开跑！

此时直接打开 ZDTY 大概率会无限闪退，不必担心，Fake Location 在重启后会有大约 180s 的 CD，此时是可以打开 ZDTY 的，虽然我到现在也没有搞清楚原理。

因此我们的步骤就是：重启后先打开 ZDTY，进入到选择的运动场所的界面，然后打开 Fake Location，待 CD 完成后启动路线模拟，再回到 ZDTY 开始运动，就可以享受运动的快乐（雾）啦！
